# WiFiWer

Herramienta automatizada de post-explotación para Windows diseñada para recuperar y mostrar todas las contraseñas de redes WiFi almacenadas en el sistema de forma rápida y legible.

### ✦ Características

*   **Recuperación Completa**: Extrae perfiles de red y claves en texto claro de todos los almacenamientos del sistema.
*   **Automatización Directa**: Evita la ejecución manual de múltiples comandos `netsh`.
*   **Formateado Limpio**: Presenta los resultados en una tabla organizada para facilitar la lectura.

### ✦ Uso

Requiere privilegios suficientes para consultar los perfiles de red. Ejecute desde una terminal:

```bash
python wifiwer.py
```
